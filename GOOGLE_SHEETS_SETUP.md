## 1. Google Sheets 생성

### 1단계: 새 스프레드시트 만들기

1. 웹 브라우저에서 [Google Sheets](https://sheets.google.com) 접속
2. 왼쪽 상단 "+ 새로 만들기" 클릭 또는 "빈 스프레드시트" 클릭
3. 스프레드시트 이름 변경: **"디스크 케어 일지 - 안미라"**

### 2단계: 시트명 변경

1. 하단의 "Sheet1" 탭 우클릭
2. "이름 바꾸기" 선택
3. 이름을 **"일일기록"** 으로 입력 (정확히 입력!)
4. Enter 키를 눌러 확인

⚠️ **중요**: 시트명은 정확히 "**일일기록**"이어야 합니다. Apps Script 코드에서 이 이름을 사용합니다.

## 2. 컬럼 헤더 설정

### 첫 번째 행에 다음 헤더를 입력하세요:

| 셀 | 컬럼명 | 설명 |
|----|--------|------|
| A1 | 타임스탬프 | 기록 저장 시간 (자동) |
| B1 | 날짜 | 기록 날짜 (사용자 입력) |
| C1 | 기상통증점수 | 기상 시 통증 (0-10) |
| D1 | 취침통증점수 | 취침 전 통증 (0-10) |
| E1 | 통증위치 | 통증 부위 (쉼표 구분) |
| F1 | 통증성격 | 통증 느낌 (쉼표 구분) |
| G1 | 주요활동 | 하루 활동 내용 |
| H1 | 악화시점 | 통증 악화된 순간 |
| I1 | 완화행동 | 통증 완화 방법 |
| J1 | 다리증상 | 다리 증상 (없음/저림/당김/힘 빠짐) |
| K1 | 약물처치 | 복용한 약 |
| L1 | 특이사항 | 기타 특이사항 |
| M1 | 하루요약 | 하루 요약 |
| N1 | 경고플래그 | 안전 경고 (자동) |
| O1 | 기록ID | 고유 ID (자동, UUID) |

### 복사하여 붙여넣기

A1 셀부터 O1 셀까지 아래 텍스트를 복사하여 붙여넣으세요:

```
타임스탬프	날짜	기상통증점수	취침통증점수	통증위치	통증성격	주요활동	악화시점	완화행동	다리증상	약물처치	특이사항	하루요약	경고플래그	기록ID
```

## 3. 서식 설정 (선택)

더 보기 좋은 스프레드시트를 만들기 위한 선택적 서식 설정입니다.

### 3.1 헤더 행 서식

1. 첫 번째 행 전체 선택 (1번 행 클릭)
2. 서식 메뉴 또는 도구바 사용:
   - **굵게**: Ctrl+B (Mac: Cmd+B)
   - **배경색**: 연한 파란색 (#E3F2FD)
   - **텍스트 정렬**: 가운데 정렬
   - **텍스트 크기**: 11pt 또는 12pt

### 3.2 컬럼 너비 조정

각 컬럼의 너비를 내용에 맞게 조정:

| 컬럼 | 권장 너비 (픽셀) |
|------|------------------|
| A (타임스탬프) | 150 |
| B (날짜) | 100 |
| C, D (통증점수) | 80 |
| E, F (위치, 성격) | 150 |
| G-M (텍스트) | 200 |
| N (경고플래그) | 120 |
| O (기록ID) | 150 |

**빠른 너비 조정**:
1. 컬럼 경계선 더블클릭 → 자동 크기 조정
2. 또는 컬럼 선택 → 우클릭 → "열 크기 조정"

### 3.3 행 고정 (스크롤 시 헤더 유지)

1. 첫 번째 행 선택
2. 보기 → 고정 → 1개 행

## 4. 데이터 검증 규칙 (선택)

Google Sheets에서 데이터 검증 규칙을 설정하여 직접 입력 시 오류를 방지할 수 있습니다.

### 4.1 통증 점수 범위 검증 (C, D 컬럼)

1. C2 셀 선택 (또는 C2:C1000 범위 선택)
2. 데이터 → 데이터 확인
3. 조건:
   - 유형: "숫자"
   - 조건: "다음 사이"
   - 최소: 0
   - 최대: 10
4. 잘못된 데이터: "입력 거부" 선택
5. "저장" 클릭

같은 방법으로 D 컬럼에도 적용.

### 4.2 다리 증상 드롭다운 (J 컬럼)

1. J2 셀 선택 (또는 J2:J1000 범위)
2. 데이터 → 데이터 확인
3. 조건:
   - 유형: "목록(범위 내)"
   - 항목 수동 입력: `없음,저림,당김,힘 빠짐`
4. 드롭다운 표시: 체크
5. "저장" 클릭

## 5. 조건부 서식 (선택)

특정 조건에 따라 셀 색상을 자동으로 변경하여 중요한 데이터를 강조할 수 있습니다.

### 5.1 통증 점수 색상 (C, D 컬럼)

**낮은 통증 (0-3): 초록색**

1. C2:D1000 범위 선택
2. 서식 → 조건부 서식
3. 조건:
   - 형식 규칙: "다음보다 작거나 같음"
   - 값: 3
4. 서식 스타일: 배경색 연한 초록색 (#D4EDDA)
5. "완료" 클릭

**중간 통증 (4-6): 노란색**

1. 같은 범위 선택
2. 조건부 서식 추가
3. 조건:
   - 형식 규칙: "다음 사이"
   - 최소: 4, 최대: 6
4. 배경색: 연한 노란색 (#FFF3CD)

**높은 통증 (7-10): 빨간색**

1. 같은 범위 선택
2. 조건부 서식 추가
3. 조건:
   - 형식 규칙: "다음보다 크거나 같음"
   - 값: 7
4. 배경색: 연한 빨간색 (#F8D7DA)

### 5.2 경고 플래그 강조 (N 컬럼)

**다리 힘 빠짐 경고**

1. N2:N1000 범위 선택
2. 조건부 서식 추가
3. 조건:
   - 형식 규칙: "텍스트 포함"
   - 값: "LEG_WEAKNESS"
4. 배경색: 빨간색 (#F44336)
5. 텍스트 색: 흰색

**심한 통증 경고**

1. 같은 범위 선택
2. 조건부 서식 추가
3. 조건:
   - 텍스트 포함: "SEVERE_PAIN_3DAYS"
4. 배경색: 주황색 (#FF9800)
5. 텍스트 색: 흰색

## 7. 공유 설정

### 본인만 편집 가능하도록 설정

1. 우측 상단 "공유" 버튼 클릭
2. 일반 액세스: **"제한됨"** (본인만 액세스 가능)
3. 또는 특정인과 공유 시:
   - 이메일 입력
   - 권한: "보기 전용" 또는 "편집자"

⚠️ **중요**: 민감한 건강 정보이므로 공유 범위를 최소화하세요.
